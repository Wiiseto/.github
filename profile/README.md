## Hi there 👋

> Currently building an app to help connect each participant of a project in the same place.
> 
Wiiseto is a Swiss-made, all-in-one cloud platform designed to simplify construction project management by connecting architects, engineers, builders, and clients in a single collaborative workspace. Currently in its early Alpha development phase, the tool helps teams seamlessly share documents, track construction progress, and communicate clearly to minimize errors on site.
